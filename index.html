Фестиваль журнала «Квантик» 2022 в Новой школе

Дата и место проведения: 29 октября 2022, суббота; Москва, Мосфильмовская улица 88, корпус 5, Новая школа, метро Раменки, Солнцевская (жёлтая) линия.

Журнал «Квантик»: kvantik.com

Новая Школа на Мосфильмовской: home.n.school

Я (Полозков Сергей Сергеевич) участвую в фестивале со своими разработками. Вот моя программа на фестивале:

  1. Волчки:
    1. Оригами с резинками
    2. Оригами из двух квадратов
    3. Круговые с зубочистками
    4. Перевёртыши из мячей для пинг-понга
    5. Перевёртыши - китайские тип-топы
  2. Вертолёты:
    1. Со скрепкой
    2. Бумага 100%
  3. Флексагоны:
    1. Тримино-трансформеры
    2. Восьмёрки двухцветные
    3. Квадратные без клея
    4. Флексотрубки кефирные прямые
    5. Флексотрубки кефирные пирамидальные
  4. Конструкторы:
    1. Магнитные гибкие
    2. Кольцевые картонные без клея
    3. Трубогранники разборные
    4. Многоугольники на резинках
  5. Авторские головоломки:
    1. Бесконечные соты
    2. Европодвесы
    3. Пирамиды из шаров
    4. Магнитный углокуб
  6. Матобъекты:
    1. Выворачиваемый тор с дыркой
    2. Ленты мёбиуса для разрезания
    3. Лестница Якова
